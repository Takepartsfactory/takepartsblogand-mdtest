# インチを英語でなんという？製造業での正しい使い方

## はじめに

製造業で日常的に使用される**「インチ」**ですが、英語での正確な発音や表記方法をご存知でしょうか？特に海外との取引やCNCプログラミングにおいて、正しい理解は非常に重要です。

本記事では、「インチ」の英語での表現から、製造現場での実際の使い方まで詳しく解説します。

## 「インチ」の英語表記と発音

### 基本情報

- **英語表記**: **inch** (単数形) / **inches** (複数形)
- **発音**: /ɪntʃ/ (インチ)
- **記号**: **in** または **"** (ダブルクォート)

### 発音のポイント

**「inch」**の発音は、日本語の「インチ」とほぼ同じですが、以下の点に注意：

- **「i」** は短い「イ」音
- **「ch」** は「チ」音（日本語とほぼ同じ）
- 単語の最後は軽く発音

## 製造業での使用例

### 寸法表記での使い方

```
正しい表記例：
• 1 inch = 1" = 25.4mm
• 0.5 inches = 0.5" = 12.7mm  
• 1/4 inch = 1/4" = 6.35mm
• 3.5 inches = 3.5" = 88.9mm
```

### CNCプログラミングでの表記

**G20コード（インチ単位）使用時：**

```gcode
G20 ; インチ単位モード
G00 X1.0 Y0.5 ; X軸1インチ、Y軸0.5インチ移動
G01 X2.5 Y1.25 F10.0 ; 送り速度10インチ/分
```

## 関連する技術英語

### よく使われる表現

**寸法関連：**
- **Diameter**: 直径 (例: 1" diameter = 1インチ径)
- **Radius**: 半径 (例: 0.5" radius = 0.5インチ半径)
- **Thickness**: 厚み (例: 0.25" thick = 0.25インチ厚)
- **Length**: 長さ (例: 4" long = 4インチ長)

**精度関連：**
- **Tolerance**: 公差 (例: ±0.001" = ±0.001インチ)
- **Precision**: 精度
- **Dimension**: 寸法

## CNCプログラミングでの注意点

### G20/G21コードの使い分け

```gcode
G21 ; ミリメートル単位（日本の標準）
G00 X25.4 Y12.7 ; 25.4mm, 12.7mm移動

G20 ; インチ単位（アメリカ標準）
G00 X1.0 Y0.5 ; 1インチ、0.5インチ移動
```

### 工具径補正での単位

```gcode
G20 ; インチモード
G41 D01 ; 工具径補正（工具テーブルの値もインチ単位）

G21 ; ミリモード  
G41 D01 ; 工具径補正（工具テーブルの値もミリ単位）
```

## よくある間違いと注意点

### 1. 複数形の使い方
- **正しい**: "3 inches long" (3インチの長さ)
- **間違い**: "3 inch long"

### 2. 記号の使い分け
- **図面**: 1.5" (ダブルクォートが一般的)
- **文章**: 1.5 in または 1.5 inches

### 3. 分数表記
- **アメリカ式**: 1/4" (クォーター・インチ)
- **小数**: 0.25" (ポイント・トゥー・ファイブ・インチ)

## 実務での活用場面

### 海外取引での図面読み取り

アメリカの図面では：
- 寸法がインチ単位
- 公差も±0.005"など
- ねじ規格もインチ系（1/4-20 UNCなど）

### 輸入工具の仕様確認

```
例：エンドミル仕様
• 径：1/2" (12.7mm)
• シャンク径：1/2" (12.7mm)  
• 全長：3" (76.2mm)
• 刃長：1" (25.4mm)
```

## まとめ

**「インチ」**の英語表現をまとめると：

1. **表記**: inch (単数) / inches (複数)
2. **発音**: /ɪntʃ/ (インチ)
3. **記号**: " または in
4. **換算**: 1" = 25.4mm

製造業では国際的なやり取りが増えているため、インチ単位の正確な理解は必須スキルです。特にCNCプログラミングや図面読み取りにおいて、単位の間違いは致命的な加工ミスにつながる可能性があります。
